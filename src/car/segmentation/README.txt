Matlab wrapper for binary segmentation + min-marginals computation using graph-cuts.

To run the example you will need the MAXFLOW code from


http://pub.ist.ac.at/~vnk/software.html

To compile it in Matlab, run mexify
, making sure that all the files are in the same directory.
